Knowledge Graph Reasoning#

This page contains benchmarks of knowledge graph reasoning methods.

We use the filtered ranking protocol for knowledge graph reasoning. For a test triplet \(\langle h, r, t\rangle\), we rank it against all negative triplets \(\langle h, r, t'\rangle\) or \(\langle h', r, t\rangle\) that do not appear in the knowledge graph. We report the mean rank (MR), mean reciprocal rank (MRR) and HITS at K (HITS@K) over the test set.
